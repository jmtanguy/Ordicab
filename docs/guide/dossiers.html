Fonctionnalités

Dossiers

Chaque dossier enregistré dans Ordicab dispose d'une fiche détaillée répartie en plusieurs onglets. Voici le tour complet des fonctionnalités disponibles.

Vue d'ensemble — le portefeuille

L'onglet Dossiers affiche l'ensemble de vos dossiers enregistrés sous forme de cartes. Chaque carte résume :

  • le nom du dossier (correspondant au nom du répertoire sur disque) ;
  • son statut : Actif, En attente, Terminé, Archivé ;
  • son type (ex. : « Contentieux civil ») ;
  • la prochaine date clé, si elle est renseignée.

Vous pouvez filtrer les dossiers par statut, les rechercher par nom et les trier par ordre alphabétique, prochaine échéance ou dernière ouverture.

Cliquez sur Ouvrir le détail pour accéder à la fiche complète d'un dossier.

Documents

L'onglet Documents liste tous les fichiers présents dans le répertoire du dossier sur votre disque. Ordicab surveille ce dossier en temps réel — tout nouveau fichier ajouté apparaît automatiquement.

Pour chaque document, vous pouvez :

  • Annoter : ajouter une description courte et des tags pour retrouver un fichier plus vite.
  • Prévisualiser : ouvrir un aperçu intégré (PDF, Word, email, texte) sans quitter Ordicab.
  • Extraire le texte : Ordicab extrait le contenu textuel du document (PDF, DOCX, EML…) pour le rendre lisible par l'IA lors d'un export. Pour les documents scannés (images, PDF non sélectionnables), Ordicab applique une reconnaissance optique de caractères (OCR) entièrement en local — aucune donnée n'est envoyée à un service externe. Le bouton Tout extraire permet de traiter l'ensemble des documents extractibles en une seule opération.
  • Ouvrir : ouvrir le fichier dans son application native.
  • Générer un document : créer un nouveau document à partir d'un modèle en utilisant les données du dossier.
Extraction de texte — clé de l'IA.
L'extraction de texte est l'étape indispensable pour que l'IA puisse lire et exploiter le contenu de vos documents. Sans extraction, l'agent ne voit que les noms de fichiers, pas leur contenu. Pensez à lancer Tout extraire avant tout export IA, en particulier sur les dossiers comportant des documents scannés — l'OCR local s'en charge automatiquement.

Contacts

L'onglet Contacts centralise toutes les personnes liées au dossier : client, adversaire, expert, notaire, huissier, etc.

Pour chaque contact, vous pouvez renseigner :

  • Identité : titre, prénom, nom, nom affiché, genre
  • Rôle dans le dossier (liste configurable dans les paramètres de l'entité)
  • Institution ou société
  • Coordonnées : adresse, téléphone, email
  • Informations personnelles : date de naissance, nationalité, profession, numéro de sécurité sociale…
  • Champs gérés spécifiques au rôle (configurés dans votre entité)
  • Un champ libre Contexte pour toute note utile sur ce contact

Les contacts sont utilisés par les macros des modèles de documents et par l'IA lors des exports.

Dates clés

L'onglet Dates clés regroupe toutes les échéances importantes du dossier : audiences, dépôts de pièces, réunions de chantier, délais légaux…

Chaque date clé comprend :

  • un libellé (ex. : « Audience de plaidoirie ») ;
  • une date ;
  • une note de contexte libre (enjeu, état d'avancement, action attendue).

La prochaine date clé à venir est mise en avant sur la carte du dossier dans le portefeuille, et les dossiers peuvent être triés par prochaine échéance.

Références clés

L'onglet Références clés stocke les identifiants importants du dossier : numéro de rôle, référence client, référence contradicteur, numéro de marché, etc.

Chaque référence est un couple libellé / valeur (ex. : RG n° / 24/01234). Les références clés sont accessibles dans les macros des modèles et dans les exports IA.

Détails du dossier

L'onglet Détails contient les métadonnées générales du dossier :

  • Statut : Actif, En attente, Terminé, Archivé
  • Type : libellé libre décrivant la nature du dossier (ex. : « Construction — VEFA »)
  • Information : résumé narratif libre — état actuel, contexte, prochaines étapes. Ce champ est particulièrement utile pour briefer rapidement l'IA.

Désenregistrer un dossier

Pour retirer un dossier d'Ordicab, utilisez le bouton Désenregistrer dans la fiche du dossier. Seules les métadonnées Ordicab (contenues dans le dossier caché .ordicab) sont supprimées. Vos documents et fichiers restent intacts à leur emplacement d'origine.
