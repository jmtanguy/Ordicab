Intelligence artificielle

Interactions IA — Vue d'ensemble

Ordicab propose quatre façons d'intégrer l'intelligence artificielle dans votre travail. Chacune répond à des besoins différents en termes de puissance, de simplicité et de confidentialité.

Les quatre modes d'interaction

Mode En un mot Accès
IA Cowork Collaboration directe avec un agent IA (Claude, Codex…) qui lit et écrit dans vos dossiers Paramètres → IA → Claude Cowork
Assistant local Chat IA intégré à Ordicab, alimenté par un modèle tournant sur votre machine via Ollama Paramètres → IA → Local (Ollama)
Assistant via API Chat IA intégré, connecté à un fournisseur cloud (OpenAI, Anthropic, Infomaniak…) via clé API Paramètres → IA → API externe
Export / Import IA Export d'un dossier structuré à ouvrir avec n'importe quelle IA Cowork, puis réimport des résultats Onglet dossier → Export IA / Import IA

Anonymisation vs pseudonymisation

Avant de comparer les modes, il est important de comprendre la distinction entre ces deux approches de protection des données — Ordicab utilise l'une d'elles de façon innovante.

Anonymisation — irréversible

L'anonymisation supprime ou masque définitivement les données personnelles. Une fois anonymisé, un document ne peut plus être rattaché à son sujet d'origine. C'est la méthode la plus protectrice sur le plan légal, mais elle rend les données inutilisables pour un traitement personnalisé — vous ne pouvez pas récupérer les vraies informations après coup.

Pseudonymisation — réversible

La pseudonymisation remplace les données identifiantes (noms, adresses, numéros…) par des alias cohérents et fictifs — par exemple Marie Martin devient Contact A, et ce même alias est utilisé partout dans le dossier exporté. L'IA travaille sur des données cohérentes mais non identifiantes.

La clé de correspondance reste sur votre machine. Ordicab peut donc restituer automatiquement les vraies données lors de l'import — les alias sont remplacés par les noms réels, les routines sont résolues, et le document final est complet et personnalisé.

L'avantage de la pseudonymisation réversible.
Vous pouvez envoyer un dossier à un service IA sans exposer les données personnelles de vos clients, obtenir un document de qualité en retour, puis le « démasquer » en un clic dans Ordicab. C'est le meilleur compromis entre utilité de l'IA et protection de la vie privée.

Quel mode choisir ?
Débutez avec l'Assistant via API si vous voulez la simplicité maximale avec une IA puissante. Choisissez l'Assistant local si la confidentialité absolue est prioritaire. Optez pour IA Cowork ou Export/Import pour des tâches complexes — création de documents, analyse en profondeur.
