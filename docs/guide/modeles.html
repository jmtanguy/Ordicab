Fonctionnalités

Modèles

Ordicab permet de constituer une bibliothèque de modèles de documents réutilisables. Chaque modèle peut contenir des routines — des balises dynamiques qui se substituent automatiquement aux données réelles du dossier lors de la génération.

Principe des routines

Une routine est une balise de la forme {{nom.champ}}. Lorsque vous générez un document à partir d'un modèle, Ordicab résout chaque routine en la remplaçant par la valeur correspondante dans le dossier sélectionné.

Exemple : si votre dossier contient un contact avec le rôle Client nommé Marie Martin, la routine {{contact.client.nomAffiche}} sera remplacée par Marie Martin dans le document généré.

Aucune donnée manquante bloquante.
Si une valeur n'est pas renseignée dans le dossier, Ordicab insère un marqueur visible à la place (ex. : [Nom manquant]). Le document reste utilisable et peut être complété manuellement.

Catégories de routines disponibles

CatégorieExemples de routines
Dossier Nom du dossier, statut, type, date d'ouverture, information générale
Contact (par rôle) Nom affiché, civilité, prénom, nom, adresse, email, téléphone, formules d'appel
Entité Nom du cabinet, adresse, téléphone, email, numéro de TVA
Dates clés Date d'un événement nommé (ex. : {{keyDate.Audience}})
Références clés Valeur d'une référence nommée (ex. : {{keyRef.RG}})
Système Date du jour, année en cours

Dans l'éditeur de modèle, le panneau Insérer une routine liste toutes les routines disponibles. Cliquez sur une routine pour l'insérer à la position du curseur.

Créer un modèle

Depuis l'onglet Modèles, cliquez sur Nouveau modèle. Deux types sont disponibles.

Modèle texte enrichi

Rédigez directement le contenu dans l'éditeur intégré d'Ordicab. La mise en forme riche (gras, italique, listes, titres) est conservée. Insérez des routines via le panneau latéral ou en saisissant {{ manuellement.

Modèle Word (.docx)

Importez un document Word existant comme source de mise en forme. Word conserve le contrôle total de la mise en page — polices, en-têtes, pieds de page, styles. Ajoutez les routines Ordicab directement dans le fichier Word avec la syntaxe {{routine}}.

Si vous souhaitez injecter du contenu géré dans Ordicab (par exemple un paragraphe supplémentaire), placez le marqueur {{app.content}} à l'endroit voulu dans le fichier Word.

Modifier une source Word.
Depuis la fiche du modèle, cliquez sur Ouvrir dans Word pour modifier la mise en forme et ajouter ou ajuster les routines directement dans le fichier source.

Générer un document

Un document peut être généré depuis plusieurs endroits :

  • Depuis la liste des modèles, en cliquant sur Générer sur un modèle.
  • Depuis l'onglet Documents d'un dossier, en cliquant sur Générer un document.

L'assistant de génération vous demande de :

  1. Sélectionner le dossier cible (si non déjà ouvert).
  2. Choisir le modèle à utiliser.
  3. Associer les contacts du dossier aux rôles utilisés dans le modèle (contact principal et éventuellement des contacts par rôle).
  4. Choisir le mode de génération : directe (le fichier est écrit immédiatement) ou relire avant enregistrement (un brouillon est préparé et vous pouvez l'ajuster avant de sauvegarder).

Le document généré est enregistré dans le répertoire du dossier et apparaît automatiquement dans la liste des documents.

Gérer la bibliothèque

Chaque modèle peut être modifié, dupliqué ou supprimé depuis la liste. La bibliothèque est commune à tous vos dossiers — un modèle créé une fois est utilisable sur n'importe quel dossier.
